# Aetheris Protocol

Binary protocol contracts for the Aetheris multiplayer engine. This repository contains the core traits, encoders, and protobuf definitions that form the backbone of the Aetheris ecosystem.

## Documentation

Full documentation for the protocol architecture can be found at: [docs.aetheris.io/protocol](https://docs.aetheris.io/protocol)

## Repository Structure

- `crates/aetheris-protocol`: Core traits and protocol definitions.
- `crates/aetheris-encoder-serde`: rmp-serde based encoder implementation.
- `crates/aetheris-encoder-bitpack`: High-efficiency bitpacking encoder (Phase 3).
